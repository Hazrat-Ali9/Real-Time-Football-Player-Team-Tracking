Real-Time-Football-Player-Team-Tracking is a computer vision and deep learning project designed to detect, track, and analyze football players and teams in real time from video footage.

Using advanced object detection, tracking algorithms, and team-color identification, the system can automatically identify players, track their movement across the field, and distinguish between opposing teams. This technology is useful for sports analytics, tactical analysis, and performance evaluation.

The project demonstrates how AI and computer vision can transform raw match footage into valuable game insights.

![alt text](image-2.png)

This project demonstrates **real-time football player detection, team classification, and event annotation** using **YOLO, Sports library, and Supervision**.

---

## Features

- **Player, Goalkeeper, Referee & Ball Detection**  
  Detect all key entities on the pitch in real-time.

- **Team Classification**  
  Classify players into teams using `sports.common.team.TeamClassifier`.

- **Tracking**  
  Maintain unique IDs for players using `ByteTrack`.

- **Frame Annotation**  
  Draw bounding boxes, labels, ellipses, and triangles for visual clarity.

- **Video Output**  
  Generates an annotated video highlighting players, teams, ball, and referees.

---

## Tech Stack

- **Computer Vision:** OpenCV, Supervision  
- **Object Detection:** YOLO (Ultralytics)  
- **Tracking:** ByteTrack  
- **Team Classification:** Roboflow Sports Library  
- **Deep Learning Framework:** PyTorch  
- **Environment:** Python 3.10, Conda
